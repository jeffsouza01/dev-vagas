<h2> :dart: Tecnologias</h2>

- Springboot
- Lombok
- Spring Security JWT Token
- BCrypt
- Swagger (Em implantação)
- Observabilidade (Em implantação)




<h2> :hammer: Como rodar a aplicação </h2>

No terminal, clone o projeto:
```
cd dev_vagas
```
```
mvn clean install
```
```
mvn spring-boot:run
```
